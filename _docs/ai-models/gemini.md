# Gemini API モデル一覧・料金

## モデル比較（100万トークンあたり・USD）

| モデル | 無料枠 | 入力単価 | 出力単価 | 用途 |
|---|---|---|---|---|
| **Gemini 2.5 Flash-Lite** | 無制限 | $0.10 | $0.40 | **最安。パース・重複チェック向き** |
| **Gemini 2.5 Flash** | 無制限 | $0.30 | $2.50 | バランス型 |
| **Gemini 3 Flash** | 無制限 | $0.50 | $3.00 | 最新 Flash |
| **Gemini 3.1 Flash-Lite** | 無制限 | $0.25 | $1.50 | 最新 Lite |
| **Gemini 2.5 Pro** | 無制限 | $1.25〜$2.50 | $10.00〜$15.00 | 高精度（Pro） |
| **Gemini 3.1 Pro** | なし | $2.00〜$4.00 | $12.00〜$18.00 | 最高精度 |

※ Pro の単価はプロンプト長（20万トークン以下/超過）で変動

## 無料枠の制限

| 項目 | 制限 |
|---|---|
| RPM（1分あたりリクエスト数） | モデルにより異なる |
| TPM（1分あたりトークン数） | モデルにより異なる |
| RPD（1日あたりリクエスト数） | モデルにより異なる |

## OshiLock での採用方針

- **パース・重複チェック・マージ**: `gemini-2.5-flash-lite`（最安）
- 精度不足の場合は `gemini-2.5-flash` に切り替え
- バッチ API を使えば通常料金の 50% 削減
