### Section 2 — 痛みの共感

**見出し:**

> 大きなライブは気づく。でも、小さな情報ほど見逃す。

**本文:**

- 推しがバラエティに出てたのを翌日知った
- 雑誌のインタビュー、発売日過ぎてから気づいた
- コラボグッズ、気づいたときには売り切れてた
- 毎日何サイトもチェックするのは、もう限界

> 好きだからこそ、全部追いたい。でも追いきれない。
> その悩み、Oshilockが解決する。

---

### Section 3 — 解決策（プロダクト紹介）

**見出し:**

> ファンが見つけて、AIが届ける。

**本文:**

ファンが集める。AIが届く。それがOshilock。

**仕組み（3ステップ）:**

```
1. ファンが情報を投稿する
   → TV出演、グッズ発売、コラボ情報…誰かが見つけたらすぐ共有

2. AIが整理する
   → 重複を消して、カテゴリ分けして、信頼度もチェック

3. あなたに届く
   → 通知で見逃さない。毎朝の「見逃しチェック」も届く
```

> もう自分で巡回しなくていい。ここだけ見ればOK。

**CTA:** 「無料枠を確保する」ボタン

### Section 5 — 差別化（なぜOshilockなのか）

**見出し:**

> 好きな気持ちを、安心して共有できる場所。

**比較:**

| 今の推し活 | Oshilock |
|---|---|
| 情報が5サイト以上に分散 | ここだけ見ればOK |
| Xで情報が流れて埋もれる | カードごとに整理される |
| 誰でも入れる → ノイズが多い | 有料だから安心できる空間 |
| 自分で毎日巡回する | ファンが集めてAIが届ける |

**補足:** ジャンルは問いません。あなたの推しを教えてください。対応ジャンルから順次ご案内します。
