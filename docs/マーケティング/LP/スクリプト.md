### Section 2 — 痛みの共感

**見出し:**

> 大きなライブは気づく。でも、小さな情報ほど見逃す。

**本文:**

- 推しがバラエティに出てたのを翌日知った
- 雑誌のインタビュー、発売日過ぎてから気づいた
- コラボグッズ、気づいたときには売り切れてた
- 毎日何サイトもチェックするのは、もう限界

> 本気で推してるのに、全部は追いきれない。
> その悩み、Oshilockが解決する。

---

### Section 3 — 解決策（プロダクト紹介）

**見出し:**

> ファンが見つけて、AIが届ける。

**本文:**

Oshilockは、推しの情報をファンみんなで集めて、AIが整理して届けるアプリ。

**仕組み（3ステップ）:**

```
1. ファンが情報を投稿する
   → TV出演、グッズ発売、コラボ情報…誰かが見つけたらすぐ共有

2. AIが整理する
   → 重複を消して、カテゴリ分けして、信頼度もチェック

3. あなたに届く
   → 通知で見逃さない。毎朝の「見逃しチェック」も届く
```

> もう自分で巡回しなくていい。ここだけ見ればOK。

---

### Section 4 — イベントカード（コア機能）

**見出し:**

> 情報がまとまる。そこで語れる。

**本文:**

推しに関する情報は「イベントカード」として整理される。

**カードのイメージ:**

```
┌──────────────────────────┐
│  📺 バラエティ出演          │
│  番組名: ○○○○            │
│  放送日: 7/20 21:00〜      │
│  ソース: 番組公式X          │
│  確度: ◎（公式確認済み）     │
│                            │
│  💬 24件のコメント           │
│  「録画予約した！」          │
│  「予告映像のリンク貼るね→」  │
└──────────────────────────┘
```

> Xだと感想が流れて埋もれる。
> Oshilockなら、そのトピックだけで盛り上がれる。

### Section 5 — 差別化（なぜOshilockなのか）

**見出し:**

> 本気で推す人だけの場所。

**比較:**

| 今の推し活 | Oshilock |
