# 追加要件

## 1. マーケティング方針

### ホワイトリスト（WL）

- リリース前にWL登録を募集し、リリース初日の過疎を防ぐ
- WL登録フォーム項目:
  - X（またはThreads）アカウント（必須）
  - 推し / ジャンル（自由記述・必須）
  - メールアドレスは不要（SNS経由で通知可能）
- WLが集まらなかった場合は、運営が初期データ（イベントカード）を手動で仕込む

### SNS発信

- X と Threads で毎日発信
- X: 拡散・バズ狙い（短文の問題提起、アンケート、開発スクショ）
- Threads: 共感・深い話（推し活あるある、ストーリー、長文）
- 投稿・プロフィールにWL登録導線を常設

---

## 2. 先着無料枠

### 仕組み

- **コミュニティ（グループ）ごとに先着100名が永久無料**
- 内部ロジックはグループ単位で管理
- ユーザーが複数コミュニティに早く参加すれば、すべて無料になり得る
- 101人目以降は通常課金（500円/月）

### マーケティング活用

- 初回リリース時: 「先着100名無料！」で集客
- 新コミュニティ追加時: 「○○コミュニティ開設！先着100名無料！」で都度マーケイベント化

### 表示

- 各コミュニティに「残り○○枠」を表示して緊迫感を出す

---

## 3. 参加フロー

### アカウント作成 → オンボーディング

```
アプリDL
  → アカウント作成（Apple Sign In / Google Sign In）
  → オンボーディング: 「推しを選ぼう」で最初のコミュニティを選択
  → ホーム画面へ
```

- オンボーディングで最低1つのコミュニティに参加させる（空のホーム画面を見せない）

### コミュニティ追加参加

```
ホーム画面
  → 「＋」ボタン or コミュニティ一覧画面
  → コミュニティを選んで参加
  → 先着100名以内なら無料 / 超えていたら課金開始
```

- ユーザー画面からいつでも追加参加可能

---

## 4. 課金モデル

### 基本

- 1コミュニティ 500円/月
- コミュニティごとに独立した課金

### 無料トライアル

- 7日間無料トライアル（全機能利用可能）
- 7日後に自動課金（App Store / Google Play アプリ内課金）
- 解約はいつでもOK

### 先着100名無料

- 先着100名はトライアルではなく**永久無料**
- コミュニティごとに100名枠を管理

### 課金タイミング

- コミュニティ参加時に課金が発生することを明示
- 追加参加時も同様（コミュニティごとに課金される旨を表示）

---

## 5. コミュニティ参加方式

- **オープン参加**（承認制ではない）
- 検索 → 参加ボタン → 即参加
- 有料（500円/月）が荒らしフィルターとして機能する
- 荒らし対策は既存の信頼スコア + BANの仕組みで対応
